 = Range =

	*,*  current line shotcut
	*;*  first number range given shotcut
	:*d or :.d*  Current line
	:*1*  First line
	:*%d*  All lines
	:*.,5d or :,5*  For the current line to line 5
	:*,+3*  Current line and more 3
	:*1,+3*  For line one to current and more 3
	:*1;+3*  For line one and more 3
